Pure stdlib

zero new deps ~200 lines to write cursor control complexity

ANSI escape codes + sys.stdout.write() + \r for in-place updates. Achieves everything described. No new dependencies — preserves the zero-dep philosophy. More code to own and maintain.

All three can deliver the B-style rolling window. The choice is: own the cursor-control code, or accept one dependency to get it (and much more) for free.
