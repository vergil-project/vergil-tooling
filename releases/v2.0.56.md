
# Release 2.0.56 (2026-05-26)

## Bug fixes

- **fix guard.sh fallback regex matching command names in filenames**
Replace grep-based regex matching against the full command string with a case statement on the first token only. Prevents false positives on filenames containing blocked command names.

- **raise fatal error when no CI checks register after poll timeout**
Increase poll timeout from 60s to 180s to accommodate GitHub performance degradation. After the poll loop expires, raise a GitHubAPIError instead of falling through to the check-watch command, which produced a confusing error when no checks existed. Ref #1148
