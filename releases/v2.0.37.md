
# Release 2.0.37 (2026-05-24)

## Chores

- **bump version to 2.0.37**

## Features

- **resolve VM template tag from vergil-vm config key**
Add vergil-vm field to IdentityConfig and Identity, parsed from the vergil-vm key in identities.toml. The template tag resolution order is now: --tag CLI flag > identity-level vergil-vm > config-level vergil-vm > generic vergil. This decouples the VM template version from the tooling version without requiring --tag on every invocation. Ref #1040
