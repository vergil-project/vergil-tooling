
# Release 2.1.112 (2026-07-04)

## Features

- **add intake --kind and default vrg-triage-create to org .github (#2125) (#2156)**
Phase 1 (epic #85). vrg-triage-create gains --kind {triage,idea,research} (default triage), setting the primary label, and now defaults its target repo to <org>/.github (org auto-detected; errors clearly if undetectable, matching vrg-epic-create). This supersedes the #2075 current-repo default so the whole org-wide intake queue lives in one place. --repo still overrides. idea/research labels already exist in labels.json.
