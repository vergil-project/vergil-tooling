
# Release 2.0.53 (2026-05-25)

## Bug fixes

- **fall back to config vergil version when no tooling marker exists**
Pre-existing VMs created before the tooling-tag marker was introduced have no
marker file, causing update_tooling to error. Add a fallback_tag parameter
that callers populate from the identity config vergil version, so updates work
on VMs created before this feature.

- **use contains instead of startswith for CD job name matching**
