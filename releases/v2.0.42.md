
# Release 2.0.42 (2026-05-24)

## Chores

- **bump version to 2.0.42**

## Features

- **inject Claude Code OAuth token during credential injection**
Add claude_token_path field to identity config and extend inject_credentials to write CLAUDE_CODE_OAUTH_TOKEN into a sourceable env file inside the VM. The login shell sources it automatically via .bashrc, enabling Claude Code inside the VM to authenticate using the host user Max subscription without interactive login.
