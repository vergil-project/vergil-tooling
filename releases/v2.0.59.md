
# Release 2.0.59 (2026-05-26)

## Bug fixes

- **detect containers via /proc/1/mountinfo overlay check**
The /.dockerenv file is Docker-specific and does not exist under nerdctl/containerd, causing vrg-validate to reject valid containers. Add a /proc/1/mountinfo overlay-root fallback that works for any OCI runtime, and drop the unused ST_IN_DEV_CONTAINER env var. Ref #1170
