
# Release 2.0.35 (2026-05-24)

## Chores

- **bump version to 2.0.35**

## Features

- **derive tooling and template version from identities.toml vergil key**
The vergil key in identities.toml specifies the ecosystem version used for both VM template fetch and vergil-tooling installation. Per-identity override supported. The --tag CLI flag overrides only the VM template version, while tooling always installs at the resolved vergil version.
