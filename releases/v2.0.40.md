
# Release 2.0.40 (2026-05-24)

## Bug fixes

- **remove unenforceable core.hooksPath audit check**
The hooks_path audit checked git config core.hooksPath, which is a post-clone setting that cannot be enforced. AI agents also trivially bypass the pre-commit gate via VRG_COMMIT_CONTEXT=1, making the check doubly ineffective.
