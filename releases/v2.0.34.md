
# Release 2.0.34 (2026-05-22)

## Bug fixes

- **route credential mkdir through bash for tilde expansion**
shell_run passes args directly without bash, so $HOME was not expanded. The mkdir created a literal $HOME directory under /tmp instead of the user home. Route through bash -c so ~ expands consistently with the shell_pipe calls.
