- **remove skip-rulesets escape hatch**
The skip-rulesets override in [github] config is removed. Rulesets are now always enforced. No repo uses this override; the repo it was designed for (mq-rest-admin-template) is marked for archival.

- **consolidate co-author config to single agent identity**
Replace per-harness co-author entries (claude, codex) with a single agent entry. The security boundary is human vs. not-human, not which AI tool produced the work.

- **update st-commit and tests for agent identity convention**
Replace per-harness co-author references (claude, codex) with the unified agent identity in all test fixtures. Update --agent help text to reference the config key.
