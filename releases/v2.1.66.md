
# Release 2.1.66 (2026-06-24)

## Features

- **fall back across nested-virt machine families on a reattach stockout (#1836) (#1841)**
* docs(off-platform): design spec for instance-family fallback on reattach (#1836)

On a reattach capacity stockout the VM apply is pinned to the data disk's zone with no recovery. Spec a curated nested-virt-safe family ladder swept within the pinned zone, with ladder-aware conformance so a fallback does not trip a spec-drift rebuild loop. Reattach-only for v1.
